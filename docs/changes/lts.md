## [6.1.0] - 2021-08-02 UTC+0800

### Added

* Added three options to the directive `waf_mode`.
    * ADV: control switch advanced rules.
    * CMN-METH: Equivalent to `head get post`.
    * ALL-METH: Any http request method will start checking.
